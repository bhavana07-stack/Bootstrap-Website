# Foodie Heaven Restaurant Website

## Overview

This project is a responsive restaurant website built using Bootstrap 5, HTML, and CSS.  
It is designed to display a modern food menu with a clean and structured layout.

The website uses Bootstrap components such as navbar, grid system, and cards to create a fully responsive and user-friendly design. It includes multiple sections like Home, Menu, Specials, and Contact.

---

## Features

- Fully responsive design using Bootstrap 5  
- Navigation bar with section links  
- Smooth scrolling between sections  
- Card-based food menu layout  
- Hover effects on menu items  
- Separate sections for Home, Menu, Specials, and Contact  
- Clean and modern UI design  
- Bootstrap grid system for layout management  

---

## Technologies Used

- HTML5  
- CSS3  
- Bootstrap 5  
- Bootstrap Grid System

## How to Run

1. Download or clone the repository  
2. Ensure all image files are in the same folder as index.html  
3. Open index.html in any web browser  
4. Use the navigation bar to navigate between sections

## Responsive Layout

### Desktop View

### Mobile View
- Automatically adjusts into a single-column layout using Bootstrap responsiveness
